# How To Use
- Open your Terminal or Else
- Type this to install git tools: "sudo apt-get install git"

- Type: "git clone https://github.com/BenMinxIDN/IPTracker.git"
- Type: "cd IPTracker"
- Type: "chmod +x IPTracker.py"
- Type: "python IPTracker.py"
